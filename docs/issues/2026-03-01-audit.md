## 一、中优先级问题 (🟡)

### 1.1 向量长度不匹配时的静默截断

**位置**: `nanobot/agent/memory.py:229`

```python
@staticmethod
def _cosine_similarity(a: list[float], b: list[float]) -> float:
    """Calculate cosine similarity."""
    dot_product = sum(x * y for x, y in zip(a, b))  # zip 会静默截断
    ...
```

**问题**: `zip()` 函数在 Python 3 中会在最短列表处停止，如果向量长度不同，会静默截断，导致相似度计算错误。

**影响**: 可能导致语义去重失效，错误地保留或删除内容。

**建议修复**:
```python
if len(a) != len(b):
    logger.warning("Embedding length mismatch: {} vs {}", len(a), len(b))
    return 0.0
```

## 二、低优先级问题 (🟢)

### 2.1 语义相似度检查失败时的静默处理

**位置**: `nanobot/agent/memory.py:221-223`

```python
except Exception as e:
    logger.warning("Semantic similarity check failed: {}", e)
    return False  # 失败时不阻止追加
```

**问题**: 异常被捕获后返回 `False`（表示"不相似"），可能导致重要内容被错误跳过。
